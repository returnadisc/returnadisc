{% if lang == 'en' %} Here you'll find answers to the most common questions about ReturnaDisc {% else %} Här hittar du svar på de vanligaste frågorna om ReturnaDisc {% endif %}

{% if lang == 'en' %} Why should I use ReturnaDisc? {% else %} Varför ska jag använda returnadisc.se? {% endif %}

{% if lang == 'en' %} ReturnaDisc is designed to facilitate the return of found discs to their owner in an anonymous, fast, and secure way. {% else %} ReturnaDisc är framtaget för att underlätta återlämning av hittade discar till ägaren på ett anonymt, snabbt och säkert sätt. {% endif %}

{% if lang == 'en' %} The benefits are: {% else %} Fördelarna är: {% endif %}

  • {% if lang == 'en' %} Anonymous handover without sharing personal information {% else %} Anonym överlämning utan att dela personlig information {% endif %}
  • {% if lang == 'en' %} The finder scans the QR code and places the disc in a safe spot until the owner picks it up {% else %} Den som hittar discen skannar QR-koden och placerar den på en trygg plats tills ägaren hämtar den {% endif %}
  • {% if lang == 'en' %} Keep the disc's value – permanent ink reduces value when selling or trading. The QR code can always be removed when you want to sell the disc {% else %} Behåll discens värde – permanent tusch sänker värdet vid försäljning eller byte. QR-koden kan alltid tas bort när du vill sälja discen {% endif %}
  • {% if lang == 'en' %} The community map works as a gathering place for lost discs, where more users increase the chance of getting your disc back {% else %} Community-kartan fungerar som en samlingsplats för borttappade discar, där fler användare ökar chansen att få sin disc återfunnen {% endif %}
  • {% if lang == 'en' %} No contact required – you avoid calls, messages, and coordinating meetings {% else %} Ingen kontakt krävs – du slipper samtal, meddelanden och att koordinera möten {% endif %}
  • {% if lang == 'en' %} No shipping needed – the disc can be left safely without packaging or postage {% else %} Ingen frakt behövs – discen kan lämnas säkert utan paketering eller porto {% endif %}
  • {% if lang == 'en' %} Save time – the process starts immediately when the QR code is scanned {% else %} Spara tid – processen startar direkt när QR-koden skannas {% endif %}
{% if lang == 'en' %} What do I do if I find an unmarked disc? {% else %} Vad gör jag om jag hittar en omärkt disc? {% endif %}
  • {% if lang == 'en' %} Go to the community map and check the location where you found the disc. {% else %} Gå till communitykartan och kontrollera platsen där du hittade discen. {% endif %}
  • {% if lang == 'en' %} If a matching disc is registered nearby, you can follow the same procedure: place the disc in a safe spot and mark it as found. {% else %} Om en matchande disc är registrerad i närheten kan du följa samma procedur: placera discen på en trygg plats och markera den som hittad. {% endif %}
  • {% if lang == 'en' %} The owner will then automatically receive an email with information about where the disc is located. {% else %} Ägaren får då automatiskt ett mejl med information om var discen finns. {% endif %}
{% if lang == 'en' %} Can I register an already lost disc? {% else %} Kan jag registrera en redan borttappad disc? {% endif %}
  • {% if lang == 'en' %} Yes. You can register a disc you have already lost by marking the location on the community map and providing a description of the disc. {% else %} Ja. Du kan registrera en disc du redan tappat bort genom att markera platsen på communitykartan och ange en beskrivning av discen. {% endif %}
  • {% if lang == 'en' %} This makes it possible for others to identify and return it if found. {% else %} Detta gör det möjligt för andra att identifiera och returnera den om den hittas. {% endif %}
{% if lang == 'en' %} What's included in BASIC? {% else %} Vad ingår i BAS? {% endif %}
  • {% if lang == 'en' %} Free use of the service with anonymous handover, photo and position {% else %} Gratis användande av tjänsten med anonym överlämning, bild och position {% endif %}
  • {% if lang == 'en' %} Register your disc on the community map to make it visible to premium members. {% else %} Registrera din disc på community-kartan för att göra den synlig för premiummedlemmar. {% endif %}
  • {% if lang == 'en' %} Print your own QR code {% else %} Skriv ut din egen QR-kod {% endif %}
{% if lang == 'en' %} What's included in Premium? {% else %} Vad ingår i Premium? {% endif %}

{% if lang == 'en' %} Premium includes all features in BASIC, plus: {% else %} Premium inkluderar samtliga funktioner i BAS, samt: {% endif %}

  • {% if lang == 'en' %} Full access to the community map with all registered discs {% else %} Full tillgång till community-kartan med samtliga registrerade discar {% endif %}
  • {% if lang == 'en' %} Early access to new features that are launched {% else %} Early access till nya funktioner som lanseras {% endif %}
{% if lang == 'en' %} Why can't BASIC users see all discs on the map? {% else %} Varför får inte BAS-användare se alla discar på kartan? {% endif %}

{% if lang == 'en' %} This is a security measure to protect the community's lost discs from dishonest individuals. Only premium members can see exact positions of lost discs, ensuring that those who have access to the information are engaged in the community. {% else %} Detta är en säkerhetsåtgärd för att skydda communityns saknade discar mot ohederliga personer. Endast premiummedlemmar kan se exakta positioner för saknade discar, vilket säkerställer att de som har tillgång till informationen är engagerade i communityn. {% endif %}

{% if lang == 'en' %} Where should the QR code be placed? {% else %} Var ska QR-koden placeras? {% endif %}

{% if lang == 'en' %} We recommend attaching the sticker on the underside of the disc. {% else %} Vi rekommenderar att du fäster klistermärket på undersidan av discen. {% endif %}

{% if lang == 'en' %} How much does a QR sticker weigh? {% else %} Vad väger ett QR-klistermärke? {% endif %}

{% if lang == 'en' %} 0.1 grams {% else %} 0,1 gram {% endif %}

{% if lang == 'en' %} Do I have to buy stickers? {% else %} Måste jag köpa stickers? {% endif %}

{% if lang == 'en' %} No, it is not mandatory. You can print your own QR code or write the ID number directly on the disc. {% else %} Nej, det är inte obligatoriskt. Du kan skriva ut din egen QR-kod eller skriva ID-numret direkt på discen. {% endif %}

{% if lang == 'en' %} Our stickers are specially designed to stick properly and withstand water, providing the best protection for your disc. {% else %} Våra klistermärken är speciellt framtagna för att sitta fast ordentligt och tåla vatten, vilket ger bäst skydd för din disc. {% endif %}

{% if lang == 'en' %} Can I have multiple QR codes? {% else %} Kan jag ha flera QR-koder? {% endif %}

{% if lang == 'en' %} Yes. When you buy a sticker package, you get QR codes with a new ID, which are linked to your account. You can have several QR codes activated simultaneously. {% else %} Ja. När du köper ett stickerspaket får du QR-koder med ett nytt ID, som kopplas till ditt konto. Du kan ha flera QR-koder aktiverade samtidigt. {% endif %}

{% if lang == 'en' %} Can I compete with the QR code on the disc? {% else %} Kan jag tävla med QR-koden på discen? {% endif %}

{% if lang == 'en' %} No. According to PDGA rules (813.01 Illegal Disc), it is not allowed to add material to the disc. {% else %} Nej. Enligt PDGA:s regler (813.01 Illegal Disc) är det inte tillåtet att addera material till discen. {% endif %}

{% if lang == 'en' %} However, it is allowed to mark the disc with permanent ink. For example, you can write the QR code's ID number and returnadisc.com directly on the disc. {% else %} Däremot är det tillåtet att märka discen med permanent tusch. Du kan till exempel skriva QR-kodens ID-nummer och returnadisc.se direkt på discen. {% endif %}

{% if lang == 'en' %} The disc must be identification-marked during competition. An unmarked disc throw results in a warning the first time, and penalty throws on repeated occasions. {% else %} Discen måste vara identifikationsmärkt vid tävling. Ett omarkerat disc kast ger varning första gången, och straffkast vid upprepade tillfällen. {% endif %}
